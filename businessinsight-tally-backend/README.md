# businessinsight-tally-backend

## Steps to setup ESLint and prettier

###### Run only first time

`npx eslint --init`

###### Run before commit

`npm run format`

###### Husky setup

`npx husky-init`
###### Migration setup

# `npx sequelize db:migrate`

###### Docker setup

`docker compose -f development.yml up --build -d db`

###### Backend start

`nodemon`

###### Swagger url

`http://localhost:8000/api-docs/#`
